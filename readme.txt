
Following features are implemented in project.

1. To store URL with unique short key and count
2. To get unique short key
3. To get count of URL hit
4. To list all URL's with pagination

Example API usage:
http://localhost:8080/storeurl?url=google.com
http://localhost:8080/get?url=google.com
http://localhost:8080/count?url=google.com
http://localhost:8080/list?page=1&size=2
